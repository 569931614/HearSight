# Add Mind Map Visualization for Video Content

## Why
Users need a visual way to understand the hierarchical structure and key concepts of video content. Currently:
1. Video summaries are displayed as plain text in the Summary tab
2. Users cannot quickly grasp the conceptual structure and relationships in video content
3. There's no interactive way to explore video content at different detail levels

A mind map visualization will help users:
- Quickly understand the main topics and subtopics of a video
- Navigate complex content more intuitively
- Focus on specific areas of interest by collapsing/expanding nodes
- Export visual summaries for offline reference

## What Changes
- Add a new "思维导图" (Mind Map) tab in the RightPanel, next to "分句" and "总结"
- Implement mind map rendering from Markdown content (generated by AI models)
- Support interactive features: zoom/pan, node collapse/expand, and export to image
- Add backend API endpoint to store and retrieve mind map data for videos

## Impact
- Affected specs: video-visualization (new), video-browsing (modified)
- Affected code:
  - Frontend: `frontend/src/components/RightPanel.tsx` (add new tab)
  - Frontend: `frontend/src/components/MindMapViewer.tsx` (new component)
  - Frontend: `frontend/src/types/index.ts` (add mind map types)
  - Frontend: `frontend/src/services/api.ts` (add mind map API calls)
  - Backend: `backend/routers/qdrant_rag.py` (add mind map endpoints)
  - Backend: `backend/models/video.py` (add mind map field if needed)

## Dependencies
- External library for mind map rendering (e.g., markmap-lib, markmap-view)
- Backend: AI model integration for generating mind map Markdown (future work, not in this change)

## Acceptance Criteria
1. A new "思维导图" tab appears in RightPanel alongside existing tabs
2. Mind map renders correctly from Markdown format data
3. Users can zoom in/out using mouse wheel or zoom controls
4. Users can pan the mind map by dragging
5. Users can collapse/expand mind map nodes by clicking on them
6. Users can export the mind map as PNG or SVG image
7. Mind map data is fetched from backend API and cached appropriately
8. Error handling: show appropriate message when mind map data is unavailable

## Out of Scope
- AI model integration for generating mind maps (will be added later)
- Real-time mind map generation during video playback
- Collaborative mind map editing
- Custom styling/theming for mind maps beyond library defaults
